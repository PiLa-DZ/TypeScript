Output-Formatting (this all about Readability)

    Output-Formatting_Options (pretty, noErrorTruncation, preserveWatchOutput)

```JSON
{
  "compilerOptions": {
    /* -----------------------------------------------------------------
       OUTPUT FORMATTING
       Controls how the compiler speaks to you in the terminal.
       ----------------------------------------------------------------- */

    "noErrorTruncation": true, // Gives you the full error message, no matter how long it is. Essential for debugging complex database types.

    "preserveWatchOutput": true, // Keeps your terminal history alive when using 'watch mode'.

    "pretty": true // Stylizes errors with colors and context for easier reading.
  }
}
```
1. pretty (
    Stylizes errors with colors and context for easier reading.
)

2. noErrorTruncation (
    Gives you the full error message, no matter how long it is. 
    Essential for debugging complex database types.
)

3. preserveWatchOutput (
    Keeps your terminal history alive when using 'watch mode'.
)
